# Portfolio Mockup (Lightweight)

This is a static mockup for quick iteration before production build.

## Primary spec (read first)
- `Tom_Zeleznock_Portfolio_Website_Master_Spec.md`

That file is the current single source of truth for:
- latest approved copy/content
- layout and interaction decisions
- implementation guidance to convert this mockup to a production website

## Open locally
- Double-click `index.html`, or
- Run a local server from this folder:
  - `python3 -m http.server 8080`
  - Open `http://localhost:8080`

## Files
- `index.html`: Main one-page portfolio mockup
- `resume.html`: Resume page mockup
- `styles.css`: Visual system and interactions
- `script.js`: Theme toggle, scroll reveals, AI hex map, discover search logic
- `Tom_Zeleznock_Portfolio_Website_Master_Spec.md`: Updated build-ready spec

## Notes
- This mockup intentionally has no build pipeline.
- The current discover search is a local keyword simulation, not final semantic search.
